# Traceability Matrix for Site Navigation

This is the requirements traceability matrix for the process of adding an item to the cart.

### Business Requirements for Site Navigation

1. the user can navigate to the homepage successfully
2. the user can navigate to the PLP using the global navigation
3. the user can navigate to the PLP from the homepage
4. the user can navigate to the PLP from a PDP
5. the user can navigate to a PDP from the PLP
6. the user can navigate to the PDP from the cart
7. the user can navigate to the cart using the cart icon
8. the user can navigate to the cart after adding an item to the cart
9. the user can navigate to checkout from the cart
10. the user can navigate to checkout after adding an item to the cart
