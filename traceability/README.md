# Traceability

This directory contains the traceability matrices for the various forms of tests for [Polymer Shop](https://shop.polymer-project.org/), an e-commerce test site.

## Navigation

In this directory you will find two subdirectories, one for manual tests and one for automated tests. Each of these subdirectories contain files with traceability matrices for each area being tested.

### Manual Tests

* Add to Cart
* Adjust Cart Quantity
* Checkout
* Remove From Cart
* Site Navigation
