# Test Cases for Navigating to the Ladies T-Shirts Product Listing Page

Navigating to the Ladies T-Shirts Product Listing Page (PLP) from elsewhere on the website

## TC-011: Navigating to Ladies T-Shirts from the Global Navigation Menu

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the Ladies T-Shirts link in the global navigation menu

### Expected Results

* the user is taken to the Ladies T-Shirts PLP

## TC-012: Navigating to Ladies T-Shirts from the Homepage Image

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the image above the Ladies T-Shirts text

### Expected Results

* the user is taken to the Ladies T-Shirts PLP

## TC-013: Navigating to Ladies T-Shirts from the Homepage Shop Now Button

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the Shop Now button below the Ladies T-Shirts text

### Expected Results

* the user is taken to the Ladies T-Shirts PLP
