# Test Cases for Navigating to the Ladies Outerwear Product Listing Page

Navigating to the Ladies Outerwear Product Listing Page (PLP) from elsewhere on the website

## TC-005: Navigating to Ladies Outerwear from the Global Navigation Menu

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the Ladies Outerwear link in the global navigation menu

### Expected Results

* the user is taken to the Ladies Outerwear PLP

## TC-006: Navigating to Ladies Outerwear from the Homepage Image

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the image above the Ladies Outerwear text

### Expected Results

* the user is taken to the Ladies Outerwear PLP

## TC-007: Navigating to Ladies Outerwear from the Homepage Shop Now Button

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the Shop Now button below the Ladies Outerwear text

### Expected Results

* the user is taken to the Ladies Outerwear PLP
