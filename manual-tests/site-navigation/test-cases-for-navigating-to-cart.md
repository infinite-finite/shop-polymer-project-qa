# Test Cases for Navigating to the Cart

Navigating to the Cart from elsewhere on the website

## TC-009: Navigating to the Cart Elsewhere on the Site

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the image of the cart in the top right corner of the page

### Expected Results

* the user is taken to the cart


## TC-012: Navigating to the Cart After Adding an Item to the Cart

### Preconditions

* the user is viewing the PDP for any item

### Testing Steps

1. click the Add to Cart button
2. in the Added to Cart modal that appears, click the View Cart button

### Expected Results

* the user is taken to the cart
