# Site Navigation Test Plan

## Introduction

This document outlines the manual testing strategy for navigating the [Polymer Shop](https://shop.polymer-project.org/) e-commerce testing website.

## Scope

Functional testing for navigating to the following locations on the site:

* Homepage
* Product Listing Page (PLP)
* Product Details Page (PDP)
* Cart
* Checkout

## Test Objectives

* Ensure all pages of the site can be accessed by the user
* Verify the user can navigate throughout the site as expected

## Test Environment

* Browser: Chrome, Firefox
* OS: Windows 11
