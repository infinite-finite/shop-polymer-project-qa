# Test Cases for Navigating to the Product Details Page

Navigating to the Product Details Page (PDP) from elsewhere on the website

## TC-006: Navigating to the PDP from a PLP

### Preconditions

* the user is on any PLP on the website

### Testing Steps

1. click on an item

### Expected Results

* the user is taken to the PDP for the item in question


## TC-007: Navigating to the PDP from the Cart Using Product Image

### Preconditions

* there are one or more items in the user's cart
* the user has navigated to the cart page

### Testing Steps

1. click on the image associated with an item in the cart

### Expected Behavior

* the user is taken to the PDP for the item in question


## TC-008: Navigating to the PDP from the Cart Using Product Description

## Preconditions

* there are one or more items in the user's cart
* the user has navigated to the cart page

### Testing Steps

1. click on the description associated with an item in the cart

### Expected Behavior

* the user is taken to the PDP for the item in question
