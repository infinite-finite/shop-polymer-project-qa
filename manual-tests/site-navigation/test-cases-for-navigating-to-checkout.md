# Test Cases for Navigating to Checkout

Navigating to Checkout

## TC-010: Navigating to Checkout from the Cart

### Preconditions

* the user has one or more items in the cart
* the user is viewing the cart

### Testing Steps

1. click the checkout button

### Expected Results

* the user is taken to the checkout page


## TC-011: Navigating to Checkout After Adding an Item to the Cart

### Preconditions

* the user is on the PDP for any item

### Testing Steps

1. click the Add to Cart Button
2. in the Added to Cart modal that appears, click the Checkout button

### Expected Results

* the user is taken to the checkout page
