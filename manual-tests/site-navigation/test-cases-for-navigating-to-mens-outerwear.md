# Test Cases for Navigating to the Men's Outerwear Product Listing Page

Navigating to the Men's Outerwear Product Listing Page (PLP) from elsewhere on the website

## TC-002: Navigating to Men's Outerwear from the Global Navigation Menu

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the Men's Outerwear link in the global navigation menu

### Expected Results

* the user is taken to the Men's Outerwear PLP

## TC-003: Navigating to Men's Outerwear from the Homepage Image

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the image above the Men's Outerwear text

### Expected Results

* the user is taken to the Men's Outerwear PLP

## TC-004: Navigating to Men's Outerwear from the Homepage Shop Now Button

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the Shop Now button below the Men's Outerwear text

### Expected Results

* the user is taken to the Men's Outerwear PLP
