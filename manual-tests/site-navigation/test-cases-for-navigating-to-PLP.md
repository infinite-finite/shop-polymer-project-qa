# Test Cases for Navigating to the Product Listing Page

Navigating to the Product Listing Page (PLP) from elsewhere on the website

## TC-002: Navigating to the PLP from the Global Navigation Menu

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the link for any category in the global navigation menu

### Expected Results

* the user is taken to the corresponding PLP for the selected category

## TC-003: Navigating to the PLP from the Homepage Image

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the image above the description of the desired category

### Expected Results

* the user is taken to the corresponding PLP for the selected category

## TC-004: Navigating to the PLP from the Homepage Shop Now Button

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the Shop Now button below the description of the desired category

### Expected Results

* the user is taken to the corresponding PLP for the selected category


## TC-005: Navigating to the PLP from the PDP

### Preconditions

* the user is viewing the PDP for any item

### Testing Steps

1. click the left-pointing arrow located in the top left corner of the page

### Expected Results

* the user is taken to the corresponding PLP for the item in question
