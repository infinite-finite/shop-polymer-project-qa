# Test Cases for Navigating to the Men's T-Shirts Product Listing Page

Navigating to the Men's T-Shirts Product Listing Page (PLP) from elsewhere on the website

## TC-008: Navigating to Men's T-Shirts from the Global Navigation Menu

### Preconditions

* the user is on any page of the website except for the cart

### Testing Steps

1. click the Men's T-Shirts link in the global navigation menu

### Expected Results

* the user is taken to the Men's T-Shirts PLP

## TC-009: Navigating to Men's T-Shirts from the Homepage Image

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the image above the Men's T-Shirts text

### Expected Results

* the user is taken to the Men's T-Shirts PLP

## TC-010: Navigating to Men's T-Shirts from the Homepage Shop Now Button

### Preconditions

* the user is on the homepage

### Testing Steps

1. in the body, click the Shop Now button below the Men's T-Shirts text

### Expected Results

* the user is taken to the Men's T-Shirts PLP
