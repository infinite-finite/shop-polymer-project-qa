# Manual Tests

This directory contains the manual tests for [Polymer Shop](https://shop.polymer-project.org/), an e-commerce test site.

## Scope

Functional testing for the following functionality:
* site navigation
  * homepage
  * Product Listing Page (PLP)
  * Product Details Page (PDP)
  * cart
  * checkout
* adding a product to the cart
* adjusting product quantity within the cart
* removing a product from the cart
* checkout

## Testing Objectives

* site navigation
  * ensure all pages of the site can be accessed by the user
  * verify the user can navigate throughout the site as expected
* adding a product to the cart
  * verify that the user can successfully add one or more items to the cart
* adjusting product quantity within the cart
  * verify that the user can increase and/or decrease the quantity of an item in the cart
* removing a product from the cart
  * verify that the user can remove an item from the cart once it has been added
* checkout
  * verify that the user can successfully check out
